Projects


SmartTask — Personal Productivity Manager

Description:

SmartTask is a web-based task manager that helps users organize, prioritize, and track daily goals efficiently. It features task categorization, reminders, and progress visualization using interactive charts. Built with React, Node.js, Express, and MongoDB, SmartTask delivers a smooth user experience with real-time updates. The app also integrates a simple AI-based suggestion system that recommends the best time slots for task completion based on user habits.

Tech Stack: React, Node.js, Express, MongoDB

SpaceScope — Explore the Universe

Description:

SpaceScope is a space-information web app designed for astronomy lovers. It fetches real-time NASA API data to display planetary details, space images, and daily discoveries. Users can browse through planets, missions, and recent cosmic news in an elegant, responsive layout. It’s designed to make space exploration accessible and fun, while showcasing clean UI and efficient API handling.

Tech Stack: HTML, CSS, JavaScript, NASA Open API
